# benzo-wallet

The consumer wallet for **Benzo** — private USDC payments on Avalanche, shielded
with eERC. Private by default; prove what you need to.

Frontend only (Vite + React + wagmi/viem + eERC SDK). Backend, contracts, and
infra live in [`Miny-Labs/benzo`](https://github.com/Miny-Labs/benzo).

See [`AGENTS.md`](./AGENTS.md) for the stack, the backend it talks to, brand
tokens, and the contribution pipeline.

Entry for the Avalanche Team1 India Speedrun — Privacy on Avalanche.
